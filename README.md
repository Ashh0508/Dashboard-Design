# Sales Dashboard Analysis – Power BI

## 📌 Project Overview

This project presents an interactive Sales Dashboard built using Power BI to analyze business performance across sales, profit, customers, states, and payment methods. The dashboard helps stakeholders monitor key metrics, identify trends, and make data-driven decisions.

## 🎯 Objective

The objective of this project is to design an interactive dashboard that provides insights into:

* Sales performance
* Profitability
* Customer behavior
* Geographic performance
* Payment preferences

## 🛠 Tools Used

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft PowerPoint

## 📊 Dashboard Features

### Overview Page

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Profit Margin KPI
* Total Customers KPI
* Sales & Profit Trend
* Revenue by Category
* Revenue by State
* Preferred Payment Methods
* Top Customers Analysis

### Sales Analysis Page

* Average Order Value
* Sales Trend Analysis
* Revenue by Category
* Payment Mode Analysis
* Product Performance Insights

### Customer Analysis Page

* Customer Spending Analysis
* Top Customers by Revenue
* Customer Profitability Analysis
* Customer Purchase Behavior

### State Analysis Page

* State-wise Sales Analysis
* State-wise Profit Analysis
* Top Performing States
* Geographic Revenue Insights

## 🔄 Interactive Features

* Date Filter
* Category Filter
* State Filter
* Payment Mode Filter
* Multi-Page Navigation using Page Navigator

## 📈 Key Insights

* Total Sales reached approximately $3M.
* Total Profit reached approximately $756K.
* Profit Margin remained healthy at 26.25%.
* Office Supplies generated the highest revenue.
* New York was the top-performing state.
* Debit Card was the most preferred payment method.
* Most customers placed only one order, indicating opportunities for customer retention strategies.

## 💡 Business Recommendations

* Increase focus on high-performing categories such as Office Supplies.
* Prioritize marketing efforts in top-performing states.
* Develop customer loyalty programs to improve repeat purchases.
* Analyze customer behavior to increase average order value.
* Continue monitoring payment preferences to optimize customer experience.
